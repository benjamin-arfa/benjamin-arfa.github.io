pybirdai

Tooling to democratize the use of BIRD as an open standard

An open-source toolkit facilitating the Banks' Integrated Reporting Dictionary (BIRD) standard for regulatory reporting. BIRD defines standardized reporting rules and templates that European banks use for providing data to supervisory bodies.

Key features

  • GUI toolkit for dictionaries & mappings
  • Docker containerization
  • Validate & transform BIRD models
  • APIs for seamless integration

Quick start

pip install pybirdai
python -m pybirdai --help

consultaition

Swiss federal consultation report generator

Automates the creation of professional Swiss federal consultation reports (Ergebnisbericht format) from structured Excel data. Vendor-agnostic LLM optimization tuned on authentic federal consultation data — no lock-in across providers.

Key features

  • Interactive Canvas editor — paragraph-by-paragraph refinement
  • REST API for programmatic report generation
  • Multi-language: German, French, Italian
  • Footnote management & export
