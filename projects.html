pybirdai

Tooling to democratize the use of BIRD as an open standard

The pybirdai project provides an open source toolkit to facilitate the use of the Banks' Integrated Reporting Dictionary (BIRD) standard for regulatory reporting. BIRD defines standardized reporting rules and templates that European banks use for providing data to supervisory bodies.

Key Features

  • • GUI toolkit for BIRD dictionaries and mappings
  • • Docker containerization for portable deployment
  • • Tools to validate and transform BIRD models
  • • APIs for seamless integration
